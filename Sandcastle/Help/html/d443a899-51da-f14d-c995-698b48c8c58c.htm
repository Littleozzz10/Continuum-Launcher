      Sprite Methods

      Methods

      Centerize Centers the Sprite around the given point.
      CheckMouse Checks to see if the mouse is hovering over the Sprite, and can check if the Left Mouse Button is clicked. Requires Initialization.
      Draw(SpriteBatch) Draws the Sprite to the given SpriteBatch.
      Draw(SpriteBatch, Boolean) Draws the Sprite to the given SpriteBatch.
      GenerateRect Returns a Rectangle representing the Sprite.
      GetCenterCoords Returns the X and Y coordinates where the Sprite would be in the center of the screen. Assumes a resolution of 1920x1080.
      GetCenterPoint Gets the Center point of the Sprite as a Vector2.
      GetSize Gets the Width and Height of the Sprite as a Vector2.
      HasTag(String) Checks if the Sprite has a tag.
      HasTag(String, OzzzFrameworkSprite) Checks if this Sprite and the parameter sprite have a tag.
      Intersects(OzzzFrameworkObjectSprite) Checks for intersection with an ObjectSprite object.
      Intersects(Rectangle) Checks for intersection with a Rectangle object.
      Intersects(OzzzFrameworkTextSprite) Checks for intersection with a TextSprite object
      JustifyRight Positions the Sprite so that the right-most edge of the Sprite is on the x-coordinate of 'rightEdge'.
      ToButton Returns the Sprite as a Button. Used for polymorphic interaction. Should only be used if the client knows the Sprite will always be a Button object.
      ToLayer Returns the Sprite as a Layer (SpriteGroup.Layer). Used for polymorphic interaction. Should only be used if the client knows the Sprite will always be a Layer object.
      ToObjectSprite Returns the Sprite as an ObjectSprite. Used for polymorphic interation. Should only be used if the client knows the Sprite will always be an ObjectSprite object.
      ToTextSprite Returns the Sprite as a TextSprite. Used for polymorphic interation. Should only be used if the client knows the Sprite will always be a TextSprite object.
      UpdatePos Updates the Position of the Sprite, applying forces in the staticForces List and applying velocity. Will usually be overriden in a concrete implementation.
