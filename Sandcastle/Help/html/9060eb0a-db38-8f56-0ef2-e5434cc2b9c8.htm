      SaveData Methods

      Methods

      AddSaveChunk Adds a SaveDataChunk to the save data to be written to file.
      AddSaveObject Adds a SaveDataObject to the save data to be written to file.
      ForceCloseStreams Forces the StreamReader and StreamWriter closed.
      ReadFile Reads in the entire save file. Deletes anything previously stored in saved data.
      SaveToFile Saves all saved data to file. Truncates the file's contents.

      Return Value

      Whether or not the file was saved in its entirety
